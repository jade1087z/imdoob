01. 변수 : 데이터 저장소

자바스크립트(JavaScript)에서 변수는 데이터를 저장하고 관리하는 데 사용되는 저장소입니다.

{
        var x = 100;                      //변수 x에 숫자 100을 저장 
        var y = 200;                      //변수 y에 숫자 200을 저장 
        var z = "javascript";             //변수 z에 문자 javascript를 저장
        
        console.log(x);
        console.log(y);
        console.log(z);
    }

var: 과거에 사용되던 변수 선언 방식으로, 현재는 잘 사용하지 않습니다.
