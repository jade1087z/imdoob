01. if()문 정리

자바스크립트에서 if 문은 조건을 검사하는 조건문입니다.

{
    // true의 조건 : true,문자열, 1, [], {}
    // false의 조건 : false, 0, null, undefind "(빈문자열)"
    
    if("조건식"){
        document.write("실행되었습니다.(true)");
    } else{
        document.write("실행되었습니다.(false)");
    }
}

if 문은 조건을 검사하고 조건에 따라 참(true) 또는 거짓(false)에 맞춰 특정 코드 블록을 실행합니다.

02. 다중if()문 정리

if()문을 사용할 때 참(true) 또는 거짓(false)외의 조건을 설정할 때 사용합니다.

{
    const num = 100;
    
    if(num == 90) {
        document.write("실행되었습니다.(num = 90)");
    } else if(num == 95){
        document.write("실행되었습니다.(num = 95)");
    } else if(num == 100){
        document.write("실행되었습니다.(num = 100)");
    } else if(num == 105){
        document.write("실행되었습니다.(num = 105)");
    } else {
        document.write("실행되었습니다.");
    }
}

else if()를 통해 참, 거짓 외의 조건을 설정합니다.

05. switch문 정리

자바스크립트(JavaScript)는 .

{
    const num = 100;
    
    switch(num){
        case 90:
            document.write("실행되었습니다.(90)");
            break;
        case 95:
            document.write("실행되었습니다.(95)");
            break;
        case 100:
            document.write("실행되었습니다(100)");
            break;
        case 105:
            document.write("실행되었습니다(105)");
            break;
        default:
            document.write("실행되었습니다.");                
    }
}
